# GLOSSARY_KR

명리학 용어를 일반 사용자가 이해하기 쉬운 표현으로 바꾼 매핑입니다.

| 명리학 용어 | 쉬운 설명 |
|---|---|
| 사주팔자 | 태어난 시점 정보를 바탕으로 본 성향 지도 |
| 명식 | 개인의 사주를 표로 정리한 구조 |
| 일간 | 나를 대표하는 중심 성향 |
| 오행 | 나무/불/흙/금속/물의 다섯 기운 |
| 음양 | 기운의 방향성(부드러움/강함, 안/밖) |
| 천간 | 겉으로 드러나는 기운 축 |
| 지지 | 생활 기반과 반복 패턴을 보여주는 축 |
| 통근 | 기운이 실제로 뿌리를 둔 상태 |
| 용신 | 균형을 잡아주는 핵심 보완 요소 |
| 기신 | 과해지면 흐름을 깨는 요소 |
| 희신 | 있으면 도움이 되는 보조 요소 |
| 구신 | 막힘을 줄여주는 해결 열쇠 |
| 억부 | 강한 기운은 누르고 약한 기운은 돕는 해석 방식 |
| 조후 | 차갑고 뜨거운 균형을 맞추는 관점 |
| 병약 | 기운의 과다/부족 상태 점검 |
| 격국 | 사주의 기본 운영 스타일 |
| 정격 | 기본 틀에 맞는 안정형 구조 |
| 변격 | 일반 틀과 다른 특수 구조 |
| 재성 | 돈/자원/현실 감각 관련 성향 |
| 관성 | 규칙/책임/직무 압력 관련 성향 |
| 인성 | 학습/회복/보호 에너지 |
| 식상 | 표현/성과/생산성 에너지 |
| 비겁 | 자기주도/경쟁/동료 에너지 |
| 정재 | 안정적 수입과 관리 성향 |
| 편재 | 기회형 수입과 확장 성향 |
| 정관 | 규범 중심 책임 수행 성향 |
| 편관 | 압박 상황 대응과 돌파 성향 |
| 정인 | 체계적 학습과 지원 성향 |
| 편인 | 직관적 학습과 독립 탐색 성향 |
| 식신 | 꾸준한 생산과 실무 성향 |
| 상관 | 창의적 표현과 문제 제기 성향 |
| 비견 | 독립성과 자기 기준 성향 |
| 겁재 | 경쟁 상황에서의 추진 성향 |
| 합 | 서로 다른 기운이 결합해 방향이 바뀜 |
| 충 | 충돌로 인해 변화가 크게 발생 |
| 형 | 마찰이 누적되어 부담이 생김 |
| 파 | 기존 구조가 느슨해지거나 깨짐 |
| 해 | 얽힘이 풀리거나 갈등이 완화됨 |
| 원국 | 태어날 때 고정된 기본 구조 |
| 대운 | 약 10년 단위의 큰 흐름 변화 |
| 세운 | 해마다 바뀌는 흐름 |
| 월운 | 월 단위의 단기 흐름 |
| 일운 | 하루 단위 컨디션 변화 |
| 공망 | 기대 대비 체감이 비어 보이는 구간 |
| 신강 | 자기 에너지가 강한 상태 |
| 신약 | 외부 영향에 민감한 상태 |
| 조열 | 뜨거운 성향이 강한 상태 |
| 한습 | 차갑거나 무거운 성향이 강한 상태 |
| 용희신 | 균형 회복에 특히 유리한 요소 묶음 |
| 태과 | 특정 성향이 지나치게 강한 상태 |
| 불급 | 특정 성향이 부족한 상태 |
| 재다신약 | 현실 이슈는 많지만 에너지 소모가 큰 구조 |
| 관다신약 | 책임 압력이 커서 피로 누적이 쉬운 구조 |
| 인다신약 | 생각/학습은 많지만 실행이 늦어질 수 있는 구조 |
