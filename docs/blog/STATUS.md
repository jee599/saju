# Velog 발행 준비 상태

업데이트 시각: 2026-02-22 17:00 KST

## 운영 규칙
- 글은 `docs/blog/drafts/`에 누적
- 한 편당 목표 분량: 5,000~7,000자 (A4 약 4장)
- 상태값
  - `DRAFTING`: 집필 중
  - `READY_FOR_REVIEW`: 발행 직전본
  - `READY_TO_PUBLISH`: 즉시 발행 가능

## 시리즈 목록
1. GPT only 버전 개발기
2. Claude only 버전 개발기
3. Hybrid(Claude 기획 + GPT 구현) 개발기

## 현재 상태
- GPT only 1편: DRAFTING
- Claude only 1편: DRAFTING
- Hybrid 1편: DRAFTING

## 알림 규칙
- 각 글이 `READY_TO_PUBLISH`가 되면 사용자에게 별도 알림 전송
