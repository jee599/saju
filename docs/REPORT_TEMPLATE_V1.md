# REPORT_TEMPLATE_V1

사주 리포트 생성의 기본 포맷과 문장 규칙입니다. 톤은 전문가형/근거형/확률형을 유지합니다.

## 1) 공통 문장 규칙
- 단정 표현 대신 확률 표현 사용: `~할 가능성이 높습니다`, `~확률이 큽니다`, `~경향이 관찰됩니다`.
- 근거 없는 공포 유도 금지: `큰일 납니다`, `반드시 실패합니다` 같은 표현 금지.
- 행동 제안은 구체적으로 제시: 기간, 우선순위, 실행 난이도 포함.
- 과도한 미신적 문구 금지: `무조건`, `절대`, `운명 확정` 금지.
- 문체: 존댓말, 짧은 문장 중심, 사실형 어조 유지.

## 2) 길이 목표
- 표준 리포트: 600~900자
- 심화 리포트: 1,800~2,600자
- 모바일(10~30초 소비) 우선: 핵심 문단은 3줄 내에서 결론 제시

## 3) 표준 리포트 템플릿 (무료/입문 유료)

### A. 요약 (2~3문장)
- 현재 흐름의 핵심
- 당장 유리한 선택 1개
- 당장 피할 선택 1개

### B. 시간축 해석
#### 과거 (회고)
- 반복된 선택 패턴 1~2개
- 강점이 발현된 맥락 1개

#### 현재
- 지금 가장 영향력이 큰 변수 1~2개
- 에너지 분산/집중 포인트

#### 가까운 미래 (4~8주)
- 가능성이 높은 전개 2가지
- 준비 시 기대되는 결과 1가지

### C. 분야별 해석
- 일/학업: 추진 타이밍, 협업 방식
- 재정: 지출 통제 포인트, 보수적 의사결정 지점
- 관계: 소통 방식, 갈등 회피보다 조정 전략
- 건강/리듬: 수면/회복/과부하 신호

### D. 실행 가이드
- 이번 주 해야 할 일 3가지
- 하지 말아야 할 일 2가지
- 체크 지표 2가지(예: 수면시간, 미루기 횟수)

## 4) 심화 리포트 템플릿 (유료)

### A. 핵심 결론
- 핵심 성향 축 2개
- 의사결정 강점/취약점 1개씩

### B. 시간축 정밀 해석
#### 과거
- 반복 패턴의 원인 가설 2개
- 성과가 높았던 조건/환경 2개

#### 현재
- 기회 신호 3개
- 리스크 신호 3개
- 현재 전략의 유지/수정/중단 판단

#### 가까운 미래 (3개월)
- 낙관/기준/보수 시나리오
- 시나리오별 대응 우선순위

### C. 분야별 심화 해석
- 커리어/사업: 역할 적합도, 레버리지 포인트
- 재정: 현금흐름 안정성, 지출 민감 구간
- 관계: 신뢰 형성 속도, 충돌 트리거
- 생활 리듬: 회복 루틴, 집중 시간대 설계

### D. 실행 가이드 (90일)
- 30일: 정리/중단할 것
- 60일: 테스트할 것
- 90일: 확장할 것
- 실패 방지 장치: 사전 체크리스트

## 5) 결과 카드용 축약 텍스트 규칙
- 한 문장 28자 내 권장
- 전문 용어는 용어집 기준 쉬운 말로 치환
- 공유 유도 문구는 정보형으로 제한: `내 성향 핵심 요약 보기`

## 6) 품질 체크리스트
- [ ] 확률형 문장 비율 60% 이상
- [ ] 행동 제안 최소 3개 포함
- [ ] 과도한 단정/공포 표현 없음
- [ ] 모바일 첫 화면에서 결론 확인 가능
