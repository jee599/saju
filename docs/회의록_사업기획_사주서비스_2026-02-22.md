- 일시: 2026-02-22 00:16 (KST)

- 범위: 서비스명, 가격, 셀링포인트, 디자인 방향, 유입/전환 전략, 실험 계획

## 0) 전제(팩트 vs 가정)
### 확인된 팩트
1. 현재 코드베이스는 웹/앱/API 모노레포 초안이 생성되어 있음.
2. 타깃은 인스타/카톡 공유 비중이 높은 10~30대.
3. 원하는 결과물 톤은:
   - 쉬운 설명
   - 확률/경향 기반 표현
   - "실제 명리학 공부한 사람이 설명하는 느낌"
4. 우선순위는 앱보다 웹 MVP 선출시.

### 가정(검증 필요)
1. 단건 결제(리포트형)가 구독보다 초기 전환이 높을 수 있음.
2. 긴 리포트(심화형) 수요가 충분할 수 있음.
3. 공유형 카드가 유료 전환 보조지표(재유입)에 기여할 수 있음.

## 1) 서비스 네이밍 후보 (1차)
> 원칙: 기억 용이성 + 신뢰감 + 공유시 어색하지 않음

### 후보 A: "사주노트"
- 장점: 기록/리포트 느낌이 강함, 문서 판매와 궁합 좋음
- 리스크: 다소 평범함

### 후보 B: "명리리포트"
- 장점: 전문성 즉시 전달
- 리스크: 입문자에게 다소 딱딱함

### 후보 C: "오늘의 명리"
- 장점: 대중 친화, SNS 카드형에 잘 맞음
- 리스크: 장문 심화 리포트 브랜딩은 약할 수 있음

### 권고
- **서비스명: 사주노트**
- **상품 라인명: 명리 리포트 / 명리 심화 리포트**

## 2) 가격 전략 (초안)
> 현재는 시장 데이터 미보유 상태. 아래는 실험 시작점(가설 가격)

### 상품 구조
1. 무료 요약 (리드 생성)
2. 단건 표준 리포트
3. 단건 심화 리포트
4. (추후) 명리+관상 통합 리포트

### 가격 앵커(초안)
- 표준: **4,900원**
- 심화: **9,900원**
- 통합(추후): **14,900원**

### 가격 실험 설계 (필수)
- A안: 3,900 / 7,900
- B안: 4,900 / 9,900
- C안: 5,900 / 11,900
- 판단지표: 결제전환율, 객단가(ARPPU), 환불률, CS불만율

## 3) 핵심 셀링포인트 (구매 사유)
1. **전문성 + 쉬운 해설 동시 제공**
   - 명리 용어를 숨기지 않고, 즉시 쉬운 언어로 번역
2. **과거-현재-가까운 미래 구조**
   - 읽는 즉시 자기 맥락에 대입 가능
3. **단정형 문장 배제, 경향/확률 기반 문장**
   - 신뢰도·수용성 강화
4. **공유 최적화 카드 자동 생성**
   - 카톡/인스타로 전파 가능한 요약 카드
5. **실행 가이드 포함**
   - "이번 주 행동 체크리스트" 제공 (읽고 끝나지 않게)

## 4) 디자인 방향 (팔리는 화면 설계)

## 톤앤매너
- 키워드: **차분함, 신뢰, 고급감, 가독성**
- 금지: 과도한 점술/미신 UI, 번쩍이는 이펙트 남발

## 비주얼 시스템
- 컬러: 다크 네이비 + 아이보리 + 골드 포인트
- 타이포: 본문 가독성 중심(모바일 우선), 제목은 세리프 보조 가능
- 컴포넌트: 카드형 리포트, 단계형 입력, 진행률 표시

## 전환 UI 원칙
1. 입력 단계는 최대 1스크린 내 완료
2. 결과는 "무료 요약 → 심화 잠금" 2단
3. 결제 CTA는 1개만 강하게 (선택 피로 감소)
4. 결제 직전 샘플 1~2문단 노출 (품질 신뢰)

## 5) 유입·바이럴 전략 (웹 MVP 기준)

## 채널 우선순위
1. 인스타 릴스/스토리
2. 카톡 공유
3. 검색 유입(롱테일 키워드)

## 공유 단위 설계

- 카드 3: 이번 주 주의 포인트

## 카피 원칙
- 자극형 단정 금지
- "가능성/경향" 표현 유지
- 과장 약속 금지

4. 가까운 미래 포인트
