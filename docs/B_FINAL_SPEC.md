## 1. 왜 B인가 — 변형 선정 근거

### 1.1 A vs B 비교 요약

| 차원 | A (GPT 기반) | **B (Claude 기반) ✓** |
|------|-------------|----------------------|
| 문체 | 실용·간결형 | **서사형·대화형 장문** |
| 설계 초점 | 전환 흐름 최적화 | **콘텐츠 품질 + 디자인 완성도** |
| UX 카피 | 기능 설명 중심 | **감정 맥락 + 행동 유도** |
| 용어 처리 | 별도 팝업/툴팁 | **문장 내 자연 삽입(인라인 교육)** |
| 브랜드 톤 | 정보 전달자 | **신뢰할 수 있는 조언자** |

### 1.2 B 선택 이유

**핵심 판단**: 사주 서비스의 본질은 "읽는 경험"이다.

사주 리포트는 결과 데이터가 아니라 **읽고, 공감하고, 행동으로 옮기는 콘텐츠**다. B 변형이 추구하는 서사형 문체와 디자인 완성도는 다음 세 가지 비즈니스 목표에 직결된다:

1. **체류 시간 → 결제 전환**: 장문 서사를 끝까지 읽은 사용자는 "더 알고 싶다"는 감정이 생긴다. 무료 구간에서 서사 몰입이 충분히 이루어져야 유료 잠금 해제 의지가 형성된다.
2. **공유 가치**: "내 이야기"처럼 읽히는 리포트는 캡처해서 공유할 확률이 높다. 기능 나열형 텍스트는 공유되지 않는다.
3. **재방문 이유**: "한 번 읽고 끝"이 아니라, 시간이 지나 다시 꺼내 읽는 리포트가 되어야 구독 전환과 리텐션의 기반이 된다.

### 1.3 B 변형의 콘텐츠 원칙 (불변 규칙)

| 원칙 | 설명 | 예시 |
|------|------|------|
| 확률형 문장 | 단정 대신 가능성·경향 표현 | "~할 가능성이 높습니다" |
| 인라인 용어 교육 | 전문 용어를 문장 속에서 자연스럽게 풀기 | "식신(아이디어를 결과물로 만드는 별)이…" |
| 행동 실행 가이드 | 읽고 끝나지 않게, 구체적 행동 3개 포함 | "이번 주 한 가지만 골라 완료해 보세요" |
| 공포 유발 금지 | 불안 자극 표현 완전 배제 | ~~"큰일 납니다"~~ → "주의가 필요한 구간입니다" |
| 이름 호칭 사용 | 리포트 전체에서 사용자 이름으로 호칭 | "지동님의 현재 흐름은…" |

## 2. 제품 비전과 포지셔닝

- **서비스명**: 사주노트 (SajuNote)
- **상품 라인명**: 명리 리포트 / 명리 심화 리포트
- **태그라인 (KR)**: "전통 명리를 확률 언어로 읽는 신뢰형 리포트"
- **태그라인 (EN)**: "Your birth chart, interpreted through probability — not superstition"

| 세그먼트 | 설명 | 핵심 니즈 |

| **실용 탐색층** (25–35세) | 진로·연애 결정 시점에 사주를 참고하고 싶음 | "신뢰할 수 있는 근거, 행동 가이드" |
| **경험 재방문층** (30–39세) | 과거에 사주를 본 적 있음, 더 정밀한 해석 원함 | "심화 리포트, 대운 타임라인" |
| **글로벌 호기심층** (EN) | K-culture 관심, 동양 철학에 호기심 | "English report, cultural context" |

### 2.3 상품 구조 & 가격

| 티어 | 내용 | 가격 | 분량 |
|------|------|------|------|
| **무료 요약** | 핵심 성향 1문단 + 현재 흐름 요약 + 즉시행동 1개 | ₩0 | 150–620자 |
| **표준 리포트** | 6개 도메인 × 시간축 + 실행 가이드 | ₩4,900 (실험 시작점) | 3,000–6,000자 |

> **가격 실험**: A안(3,900/7,900), B안(4,900/9,900), C안(5,900/11,900) — 4주간 병렬 실험 후 확정

### 2.4 핵심 셀링포인트

1. **"명리 전문가가 쉽게 설명해 주는 느낌"** — 용어를 숨기지 않되, 문장 속에서 즉시 번역
2. **"과거-현재-미래 3축 해석"** — 내 과거 패턴을 인지하고, 지금 할 일을 명확히 알 수 있음
3. **"확률 기반이라 믿을 수 있음"** — 단정·공포 표현 제로, 경향과 가능성으로만 서술
4. **"읽고 끝나지 않는 리포트"** — 이번 주 행동 체크리스트 포함
5. **"공유하면 자연스러운 카드"** — 카카오/인스타 최적화 공유 카드 3종 자동 생성

## 3. 이중언어(KR/EN) UX 전략

### 3.1 언어 전략 원칙

| 원칙 | 설명 |
|------|------|
| **KR-first, EN-parallel** | 한국어가 1차 언어, 영어는 동시 지원하되 한국어 품질을 기준으로 영어를 맞춤 |
| **문화 번역 > 직역** | 명리 용어는 직역하지 않고, 영어 사용자가 이해할 수 있는 문화적 맥락을 함께 제공 |
| **UI 레이블은 짧게** | 버튼·메뉴 등 UI 요소는 KR/EN 모두 최소 글자 수로 유지 |
| **리포트는 별도 톤** | KR 리포트는 존댓말 대화체, EN 리포트는 friendly expert 톤 |

### 3.2 언어 감지 & 전환

```
기본 동작:
1. 브라우저 Accept-Language 헤더 감지
2. ko, ko-KR → 한국어 UI + 한국어 리포트
3. en, en-* → 영어 UI + 영어 리포트
4. 그 외 → 영어 fallback
5. 사용자가 수동 전환 가능 (헤더 언어 토글)
```

## 4. 웹 퍼널 IA·카피 설계

### 4.1 퍼널 전체 흐름

```
[1] 랜딩 → [2] 입력 폼 → [3] 무료 결과 → [4] 페이월 → [5] 결제 → [6] 전체 리포트 → [7] 공유/재방문
     ↓            ↓              ↓             ↓            ↓             ↓               ↓
  인지/신뢰     참여/입력      가치 확인      욕구 증폭    전환 완료     만족/몰입      확산/리텐션
```

### 4.2 각 단계별 IA + 카피 + 전환 요소

#### [1] 랜딩 페이지 (`/`)

**목적**: 서비스 인지 + 신뢰 확보 + 입력 유도

| 섹션 | 구성 요소 | KR 카피 | 전환 요소 |
|------|----------|---------|----------|
| 히어로 | 제목 + 부제 + CTA | "사주노트 — 내 사주를 확률로 읽다" | [무료로 내 사주 보기] 버튼 |
| 셀링포인트 | 아이콘 3개 | ①쉬운 해설 ②확률 기반 ③행동 가이드 | 각 포인트 클릭 시 스크롤 |
| 샘플 리포트 | 미리보기 카드 | 실제 리포트 2-3문단 발췌 | "이런 리포트를 받게 됩니다" |
| 가격 안내 | 티어 비교 | 무료/표준/심화 3단 | 가격 옆 [시작하기] CTA |
| FAQ | 아코디언 | "정말 무료인가요?" 등 5개 | 불안 해소 |
| 하단 CTA | 고정 배너 | "30초면 충분합니다" | [무료 분석 시작] |

**모바일 최적화 규칙**:
- 히어로 CTA는 첫 화면(above the fold)에 노출
- 스크롤 깊이 50% 이상 시 하단 고정 CTA 활성화

#### [2] 입력 폼 (`/free-fortune`)

**목적**: 최소 마찰로 정보 수집, 1스크린 완료

| 구성 | 상세 |
|------|------|
| 폼 레이아웃 | 단일 컬럼, 5개 필드, 1스크린 내 모두 노출 |
| 진행 표시 | 상단 스텝 인디케이터 (1/3: 정보 입력 → 2/3: 분석 중 → 3/3: 결과) |
| 이름 필드 | 2자 이상, 실시간 유효성 검사 |
| 생년월일 | 날짜 선택기 + 직접 입력 병행 |
| 태어난 시간 | "(선택) 시간을 모르면 비워두셔도 됩니다" — 불안 해소 카피 |
| 성별 | 라디오 버튼 3개 (남성/여성/기타) |
| 양력/음력 | 토글 스위치, 기본값 양력 |
| 제출 버튼 | "내 사주 분석하기" — 로딩 시 "분석 중입니다…" 애니메이션 |

**KR 마이크로카피**:
- 폼 상단: "30초면 충분합니다. 정확한 결과를 위해 생년월일을 확인해 주세요."
- 시간 필드 아래: "태어난 시간을 알면 더 정밀한 해석이 가능합니다."
- 제출 전 안내: "입력하신 정보는 분석에만 사용되며, 서버에 저장되지 않습니다."

**EN 마이크로카피**:
- 폼 상단: "Just 30 seconds. Please double-check your birth date for accuracy."
- 시간 필드 아래: "Knowing your birth time enables a more precise reading."
- 제출 전 안내: "Your info is used for analysis only and is not stored on our servers."

#### [3] 무료 결과 (`/result`)

**목적**: 가치 증명 + 유료 전환 욕구 형성

## 10. 부록 — 용어 매핑, 금지 표현, 참조 문서

### 10.1 핵심 용어 인라인 설명 풀 (리포트 삽입용)

리포트 엔진은 아래 풀에서 도메인에 맞는 용어를 선택하여 문장 속에 자연 삽입한다.

| 용어 | 쉬운 설명 | 실용 맥락 |
|------|----------|----------|
| 식신 | 아이디어를 결과물로 만드는 생산의 별 | 루틴을 쌓아 성과를 만들 때 강해집니다 |
| 상관 | 표현력과 개선 감각 | 문제점을 빨리 발견하고 방식 자체를 바꿉니다 |
| 정재 | 안정적 수입과 자산 관리 성향 | 계획한 대로 수입이 들어올 때 심리적으로 안정됩니다 |
| 편재 | 기회형 수입과 확장 성향 | 새로운 기회가 보이면 빠르게 움직이는 편입니다 |
| 정관 | 규범 중심의 책임 수행 | 조직 안에서 신뢰를 쌓을 때 존재감이 올라갑니다 |
| 편관 | 압박 상황 돌파 능력 | 긴장감이 높을 때 오히려 집중력이 올라갑니다 |
| 정인 | 체계적 학습과 지원 에너지 | 배울 수 있는 환경에서 회복력이 좋아집니다 |
| 편인 | 직관적 학습과 독립 탐색 | 혼자 깊게 파고드는 시간이 필요합니다 |
| 비견 | 독립성과 자기 기준 | 자기 방식대로 할 때 효율이 최대입니다 |
| 겁재 | 경쟁 에너지와 추진력 | 경쟁 구도에서 에너지가 급상승합니다 |
| 대운 | 10년 단위 큰 흐름 | 중장기 선택의 타이밍을 읽는 프레임입니다 |
| 세운 | 해마다 바뀌는 흐름 | 올해의 키워드를 잡아주는 단위입니다 |

> **전체 60+ 용어**는 `/docs/GLOSSARY_KR.md` 참조

### 10.2 금지 표현 목록

아래 표현은 리포트·UI·마케팅 카피 어디에서도 사용하지 않는다.

| 금지 유형 | 예시 | 대체 표현 |
|----------|------|----------|
| 절대적 단정 | "반드시 ~합니다" | "~할 가능성이 높습니다" |
| 공포 유발 | "큰일 납니다", "위험합니다" | "주의가 필요한 구간입니다" |
| 운명 확정 | "운명입니다", "피할 수 없습니다" | "이런 경향이 관찰됩니다" |
| 미신적 표현 | "무조건", "절대", "틀림없이" | "~확률이 큽니다" |
| 의학적 단정 | "병이 생깁니다" | "건강 관리에 신경 쓸 구간입니다" |
| 재무적 단정 | "돈을 잃습니다" | "지출 관리가 유리한 시기입니다" |
| 관계 파괴 | "헤어집니다", "이혼합니다" | "관계 조율이 필요한 흐름입니다" |

### 10.3 확률형 문장 패턴 라이브러리

리포트 생성 시 반복 활용되는 문장 패턴:

```
긍정적 경향:
- "~할 가능성이 높습니다"
- "~확률이 큽니다"
- "~에 유리한 흐름입니다"
- "~쪽으로 에너지가 집중되는 경향이 있습니다"

주의 필요:
- "~에 주의가 필요한 구간입니다"
- "~가 과해질 수 있는 시기입니다"
- "~보다는 ~를 우선하는 것이 유리합니다"

행동 제안:
- "이번 주는 ~에 집중해 보세요"
- "당분간은 ~을 미루는 것이 현명합니다"
- "~를 한 가지로 좁히고 완료 기준부터 정해 보세요"

조건부 해석:
- "환경에 따라 표현 방식은 달라질 수 있습니다"
- "이 패턴이 강하게 나타나는 조건은 ~입니다"
- "~한 상황에서는 다르게 작용할 수 있습니다"
```
