## 1. 수익화 전략

### 1-1. 현재 가격 적정성 분석

현재 단일 가격: 한국 ₩5,900 / 글로벌 $4.99.

한국 시장 기준으로 ₩5,900은 경쟁사 대비 중간대다. 만세력 무료, 사주카페 ₩3,000~₩10,000, 전문 상담 ₩30,000+. 문제는 "AI가 쓴 리포트에 5,900원을 낼 이유"를 증명하지 못한 상태라는 점이다. 가격 자체보다 가치 인지가 부족하다.

글로벌 $4.99는 동남아 시장에서 높다. 태국 평균 디지털 콘텐츠 구매 단가 $1~$3, 베트남 $1~$2 수준이다. 일본은 $4.99가 오히려 싸게 느껴질 수 있다(점술 1회 상담 ¥1,000~¥3,000).

**결론**: 한국 ₩5,900은 유지하되, 글로벌은 국가별 차등 가격이 필수다.

### 1-2. 국가별 차등 가격 전략 (PPP 기반)

Paddle이 PPP(구매력 평가) 기반 가격 설정을 지원한다. 다음 기준으로 설정:

| 국가 | 가격 | 근거 |
|------|------|------|
| 한국 | ₩5,900 | Toss 결제, 현행 유지 |
| 일본 | ¥700 (~$4.99) | 점술 시장 성숙, 높은 지불 의사 |
| 미국/영어권 | $4.99 | 기준 가격 |
| 중국(대만) | NT$150 (~$4.50) | 풍수/사주 문화권, 중상위 |
| 태국 | ฿99 (~$2.79) | PPP 56% 할인 |
| 베트남 | ₫59,000 (~$2.29) | PPP 54% 할인 |
| 인도네시아 | Rp39,900 (~$2.49) | PPP 50% 할인 |
| 인도 | ₹199 (~$2.29) | PPP 54% 할인 |

**실행**: Paddle 대시보드에서 국가별 override 가격 설정. 2주 내 적용 가능.

### 1-3. 추가 수익원

**궁합 리포트 (1개월 내 출시 목표)**
- 가격: ₩7,900 / $5.99~$6.99
- 두 사람의 사주를 비교 분석. 연애/결혼/비즈니스 궁합
- 바이럴 효과 극대화: "너도 해봐" 공유 자연스러움
- 예상 전환율: 메인 리포트 구매자의 15~20%가 추가 구매

**일일운세 구독 (3개월 내 출시 목표)**
- 가격: 월 ₩3,900 / $2.99
- 매일 아침 푸시/이메일로 오늘의 운세 + 행동 가이드
- LTV 극대화: 단건 ₩5,900 vs 구독 연 ₩46,800
- 이탈 방지: 7일 무료 체험, 첫 달 50% 할인

**기업용 API (6개월 내 탐색)**
- 미디어/커뮤니티 사이트에 사주 위젯 제공
- 가격: 월 $49~$199 (API 호출 1,000~10,000건)
- 타깃: 운세 콘텐츠를 원하는 포털, 매거진, 앱

### 1-4. 가격 A/B 테스트 계획

**1차 테스트 (즉시 실행)**
- 변수: 한국 가격 ₩3,900 vs ₩5,900 vs ₩7,900
- 트래픽 분배: 33%/33%/33%
- 최소 표본: 각 그룹 100명 방문자 (유의미한 결과까지 약 2~4주)
- 판단 지표: 결제 전환율 × 객단가 = ARPU 최대화 기준
- 도구: Vercel Edge Config + 쿠키 기반 그룹 할당

**2차 테스트 (1차 완료 후)**
- 앵커링 효과: "원래 ₩9,900 → 지금 ₩5,900" 할인 표시 vs 단일 가격
- 번들: 사주 + 궁합 ₩9,900 vs 개별 구매 합산 ₩13,800

## 3. 전환율 최적화 (CRO)

### 3-1. 현재 퍼널과 예상 이탈률

```
홈 방문 (100%)
  → 사주 입력 시작 (60%) — 이탈 40%: CTA 불명확, 신뢰 부족
    → 입력 완료 (45%) — 이탈 25%: 생시 모름, 입력 귀찮음
      → 무료 분석 확인 (42%) — 이탈 7%: 로딩 시간, 에러
        → 페이월 도달 (35%) — 이탈 17%: 무료 결과만으로 만족
          → 결제 시도 (5%) — 이탈 86%: 가격 저항, 결제 번거로움
            → 결제 완료 (3.5%) — 이탈 30%: 결제 실패, 이탈
              → 리포트 확인 (3.3%) — 이탈 6%: 이메일 미확인
```

**현실적 전환율 예측: 방문자 대비 결제 완료 2~4%**

### 3-2. 단계별 개선 방안

**홈 → 입력 시작 (목표: 60% → 75%)**
- 소셜 프루프 추가: "지금까지 N명이 분석 받았습니다" (실시간 카운터)
- 입력 폼 미리보기: 홈에서 생년월일 입력 필드를 바로 노출
- 로딩 없이 즉시 시작되는 느낌 제공

**입력 시작 → 완료 (목표: 75% → 85%)**
- "생시 모름" 옵션 더 눈에 띄게. 현재도 있지만 망설이는 사용자가 많음
- 입력 단계를 3단계 프로그레스바로 시각화
- 자동완성: 양력/음력 전환 시 즉시 반영

**무료 분석 → 페이월 (목표: 83% → 90%)**
- 무료 분석에서 유료 콘텐츠의 "맛보기" 노출:
  - "당신의 재물운 분석이 준비되었습니다" (잠금 아이콘 + 1줄 티저)
  - 5개 섹션 중 2개만 공개, 나머지 3개는 블러 처리
- 스크롤 유도: 무료 콘텐츠 끝에 자연스럽게 페이월 연결

**페이월 → 결제 (목표: 14% → 25%)**
- 이것이 핵심 병목이다
- 시간 제한 할인: "30분 내 구매 시 30% 할인" (긴급성)
- 보장 문구: "만족하지 않으면 전액 환불" (리스크 제거)
- 결제 수단 다양화: 한국은 Toss 외 카카오페이, 네이버페이 추가 검토
- 가격 앵커: 심화 ₩9,900을 먼저 보여주고, 표준 ₩5,900이 합리적으로 느껴지게

**결제 시도 → 완료 (목표: 70% → 85%)**
- 결제 페이지 로딩 속도 최적화
- 결제 실패 시 즉시 재시도 안내 + 대체 결제 수단 제안

### 3-3. 무료→유료 전환 핵심 전략

무료 분석이 "충분히 좋지만 더 알고 싶게" 만들어야 한다.

1. **미완성의 완성**: 무료 분석에서 "당신에게 특히 주의가 필요한 시기가 있습니다"처럼 호기심을 유발하는 문장을 삽입하고, 상세 내용은 유료에서만 제공
2. **구체적 수치 차이**: 무료는 "재물운이 좋은 편입니다", 유료는 "3월~5월에 투자 기회가 올 확률이 높으며, 특히 4월 둘째 주가 핵심 시기입니다"
3. **시각적 차이**: 무료는 텍스트만, 유료는 차트/그래프/타임라인 포함
4. **레퍼런스 리포트**: 무료 분석 하단에 "유료 리포트 샘플 미리보기" 링크 제공

### 3-4. 리타겟 이메일 최적화

현재 결제 미완료 사용자에게 1회 이메일 발송 중.

**개선된 이메일 시퀀스:**

| 시점 | 제목 | 내용 | 목표 |
|------|------|------|------|
| 1시간 후 | "분석이 아직 기다리고 있어요" | 무료 분석 요약 + CTA | 즉시 복귀 |
| 24시간 후 | "오늘의 운세: [한 줄 운세]" | 가치 제공 + 할인 코드 10% | 가치 인지 |
| 72시간 후 | "마지막 기회: 30% 할인" | 시간 제한 할인 | 긴급성 |
| 7일 후 | "[이름]님의 이번 달 운세 변화" | 콘텐츠형 + 소프트 CTA | 장기 관계 |

**A/B 테스트 항목:**
- 발송 시간: 오전 8시 vs 오후 9시
- 제목: 이름 포함 vs 미포함
- 할인: 10% vs 20% vs 할인 없음
- 목표 오픈율: 25% 이상, 클릭률: 5% 이상
